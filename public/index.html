Toys Table

#Method URL Body Description
GET /toys none Get a list of all the toys, will present 10 results per page, if you want to see the list of the toys.
GET /toys/search?s=doll none search for a toy by searching follwing name or info after the "s=", it will present 10 results per page, if you want to see the list of the toys.
GET /toys/category/:catName none Get a list of all the toys in thatcategory, it will present 10 results per page, if you want to see the list of the toys.
GET /toys/byId/:idGet none Get the toy with that id, it will present 10 results per page, if you want to see the toy.
